ArchLinux2021安装记录

Posted on 2021-03-18 Edited on 2021-03-19

Archlinux2021 安装教程

一、BIOS设置

禁用Safe Boot，开启Legacy Boot，按F9（各电脑要求不完全一致）进入启动项管理，选择Archlinux镜像进入。

二、连接网络

WiFi链接方式如下:

  1. 查询网卡设备名

    station wlan0 scan
    station wlan0 get-networks
    station wlan0 connect wifiname

    ping www.baidu.com

    timedatectl set-ntp true

    reflector -c China -a 6 –sort rate –save /etc/pacman.d/mirrorlist

    vim /etc/pacman.d/mirrorlist

    pacman -Syy

    fdisk -l

    fdisk /dev/nvme0n1/

    mkfs.fat -F32 /dev/nvme0n1p1
    mkfs.ext4 /dev/nvme0n1p2
    mkfs.ext4 /dev/nvme0n1p3

    mount /dev/nvme0n1p3 /mnt
    mkdir /mnt/boot
    mount /dev/nvme0n1p2 /mnt/boot
    mkdir /mnt/boot/EFI
    mount /dev/nvmeon1p1 /mnt/boot/EFI

    pacstrap -i /mnt base base-devel linux linux-firmware

    genfstab -U /mnt >> /mnt/etc/fstab

    cat /mnt/etc/fstab

    arch-chroot /mnt /bin/bash

    vim /etc/locale.gen

    locale-gen

    echo LANG=en_US.UTF-8 > /etc/locale.conf

    ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

    hwclock –systohc –utc

    echo arch > /etc/hostname

    pacman -S vim

    vim /etc/hosts

    127.0.0.1 localhost
    ::1 localhost
    127.0.1.1 arch.localdomain arch

    passwd

    useradd -m -G wheel yourname
    passwd yourname

    pacman -S sudo
    vim /etc/sudoers

    //声音驱动
    pacman -S alsa-utils
    //触摸板驱动
    pacman -S xf86-input-synaptics
    //CPU微码
    pacman -S amd-ucode
    //安装AMD集显驱动
    pacman -S xf86-video-amdgpu
    //Nvidia独显驱动
    pacman -S nvidia nvidia-utils
    //网络驱动(nm和dhcpcd现在有冲突)
    pacman -S iw netctl wpa_supplicant dialog dhcpcd networkmanager

    pacman -S ntfs-3g os-prober

    pacman -S grub efibootmgr

    grub-install –target=x86_64-efi –efi-directory=/boot/EFI –bootloader-id=Arch –recheck

    grub-mkconfig -o /boot/grub/grub.cfg

    exit
    umount -a
    reboot

    nmtui

    pacman -S xorg

    pacman -S plasma packagekit-qt5 konsole dolphin plasma-nm

    systemctl enable sddm
    systemctl enable NetworkManager
    ```

  2. 重启进入桌面环境

Tomcat7相关问题

Tomcat7插件

1.启动失败

  1. Error Message:Failed to execute goal org.apache.tomcat.maven:tomcat7-maven-plugin:2.2:run (default-cli) on project

  2. pom.xml中servlet和servlet-jsp无scope属性，属性需要设置为provided

2.版本号丢失

  1. Error Message:’build.plugins.plugin.version’ for org.apache.maven.plugins:maven-compiler-plugin is missing

  2. maven-compiler-plugin的version属性未写入

3.Encoding未设置

  1. Error Message:Using platform encoding (UTF-8 actually) to copy filtered resources, i.e. build is platfor

  2. Maven 打包时有标题中警告，需要在pom.xml文件中添加
