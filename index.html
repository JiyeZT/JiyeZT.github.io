Archlinux2021 安装教程

一、BIOS设置

禁用Safe Boot，开启Legacy Boot，按F9（各电脑要求不完全一致）进入启动项管理，选择Archlinux镜像进入。

二、连接网络

WiFi链接方式如下:

  1. 查询网卡设备名

  2. 扫描无线网并连接（假设网卡为wlan0，wifi为Wifiname）

  3. 测试连接

    ctrl+C退出测试，网线可以直接进行测试

三、修改软件源

  1. 同步系统时间(双系统情况下由于UTC设置的不一致，导致时间不同步，进入系统后再修正)

  2. 使用reflector来获取速度最快的6个镜像，并将地址保存至/etc/pacman.d/mirrorlist（未测试过，仅为备忘）

  3. 手动修改/etc/pacman.d/mirrorlist文件

    输入/China 查找到中国源，按yy复制，然后按gg跳到文本开头 按p粘贴。

  4. 更新源

四、使用fdisk分区

  1. 查看存储设备情况

  2. 选择要安装进的硬盘，进行分区（假设为nvme0n1）

    进入分区页面，基础命令如下

    n ： 新建
    t ： 修改分区格式
    p ： 查看分区情况
    w ： 写入分区修改

  3. 分区方案

    分区1 -> 200M -> /boot/EFI
    分区2 -> 200M -> /boot
    分区3 -> 150G -> /mnt

  4. 格式化分区

  5. 挂载分区

Tomcat7相关问题

Tomcat7插件

1.启动失败

  1. Error Message:Failed to execute goal org.apache.tomcat.maven:tomcat7-maven-plugin:2.2:run (default-cli) on project

  2. pom.xml中servlet和servlet-jsp无scope属性，属性需要设置为provided

2.版本号丢失

  1. Error Message:’build.plugins.plugin.version’ for org.apache.maven.plugins:maven-compiler-plugin is missing

  2. maven-compiler-plugin的version属性未写入

3.Encoding未设置

  1. Error Message:Using platform encoding (UTF-8 actually) to copy filtered resources, i.e. build is platfor

  2. Maven 打包时有标题中警告，需要在pom.xml文件中添加
