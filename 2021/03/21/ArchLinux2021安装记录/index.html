Posted on 2021-03-21

Archlinux2021 安装教程

一、BIOS设置

禁用Safe Boot，开启Legacy Boot，按F9（各电脑要求不完全一致）进入启动项管理，选择Archlinux镜像进入。

二、连接网络

WiFi链接方式如下:

  1. 查询网卡设备名
  2. 扫描无线网并连接（假设网卡为wlan0，wifi为Wifiname）
  3. 测试连接 ctrl+C退出测试，网线可以直接进行测试

三、修改软件源

  1. 同步系统时间(双系统情况下由于UTC设置的不一致，导致时间不同步，进入系统后再修正)
  2. 使用reflector来获取速度最快的6个镜像，并将地址保存至/etc/pacman.d/mirrorlist（未测试过，仅为备忘）
  3. 手动修改/etc/pacman.d/mirrorlist文件 pacman -Syy fdisk -l fdisk /dev/nvme0n1/ mkfs.fat -F32 /dev/nvme0n1p1
    mkfs.ext4 /dev/nvme0n1p2
    mkfs.ext4 /dev/nvme0n1p3 mount /dev/nvme0n1p3 /mnt
    mkdir /mnt/boot
    mount /dev/nvme0n1p2 /mnt/boot
    mkdir /mnt/boot/EFI
    mount /dev/nvmeon1p1 /mnt/boot/EFI pacstrap -i /mnt base base-devel linux linux-firmware genfstab -U /mnt >> /mnt/etc/fstab cat /mnt/etc/fstab arch-chroot /mnt /bin/bash vim /etc/locale.gen locale-gen echo LANG=en_US.UTF-8 > /etc/locale.conf ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime hwclock –systohc –utc echo arch > /etc/hostname pacman -S vim vim /etc/hosts 127.0.0.1 localhost
    ::1 localhost
    127.0.1.1 arch.localdomain arch passwd useradd -m -G wheel yourname
    passwd yourname pacman -S sudo
    vim /etc/sudoers //声音驱动
    pacman -S alsa-utils
    //触摸板驱动
    pacman -S xf86-input-synaptics
    //CPU微码
    pacman -S amd-ucode
    //安装AMD集显驱动
    pacman -S xf86-video-amdgpu
    //Nvidia独显驱动
    pacman -S nvidia nvidia-utils
    //网络驱动(nm和dhcpcd现在有冲突)
    pacman -S iw netctl wpa_supplicant dialog dhcpcd networkmanager pacman -S ntfs-3g os-prober pacman -S grub efibootmgr grub-install –target=x86_64-efi –efi-directory=/boot/EFI –bootloader-id=Arch –recheck grub-mkconfig -o /boot/grub/grub.cfg exit
    umount -a
    reboot nmtui pacman -S xorg pacman -S plasma packagekit-qt5 konsole dolphin plasma-nm systemctl enable sddm
    systemctl enable NetworkManager
    ```
  4. 重启进入桌面环境
